# 智投 AI — 全方位 AI 投資分析系統

一套整合即時股票資料與 AI 大型語言模型的投資輔助工具。透過網頁介面，讓使用者不需要寫程式，就能完成個股分析、估值計算、技術指標解讀、投資組合健診、回測模擬與月度市場摘要等工作。

---

## 系統需求

- Node.js 18 版以上
- Anthropic API Key（Claude）或 OpenAI API Key（GPT-4o），二擇一即可，兩個都有的話可以在介面上自由切換

---

## 本地安裝與啟動

第一步，進入專案資料夾並安裝相依套件：

```
npm install
```

第二步，在專案根目錄建立 `.env` 檔案，填入你的 API Key：

```
ANTHROPIC_API_KEY=你的 Claude API Key
OPENAI_API_KEY=你的 OpenAI API Key
```

兩個 Key 只需填其中一個，系統會自動偵測哪個可用。

第三步，啟動伺服器：

```
npm start
```

啟動後開啟瀏覽器，前往 `http://localhost:3000` 即可使用。

## 部署到 Render（免費雲端主機）

Render 是一個免費的雲端部署平台，可以讓你的應用程式有對外的網址，讓其他人也能使用。

1. 將專案上傳到你的 GitHub Repository
2. 前往 render.com 註冊並登入
3. 點擊右上角 New，選擇 Web Service
4. 選擇你的 GitHub Repository，點擊 Connect
5. 填入以下設定：
   - Build Command：`npm install`
   - Start Command：`npm start`
   - Instance Type：Free（免費方案）
6. 往下滑找到 Environment Variables，新增以下變數：
   - `ANTHROPIC_API_KEY`：你的 Claude API Key
   - `OPENAI_API_KEY`：你的 OpenAI API Key（選填）
7. 點擊 Create Web Service，等待約 2 至 3 分鐘完成部署

部署完成後，Render 會提供一組網址，例如 `https://your-app-name.onrender.com`。

注意：Render 免費方案在無人使用超過 15 分鐘後會進入休眠，下次有人開啟時需要等候約 30 秒喚醒。如果需要全天候運作，可升級至付費方案（約每月 7 美元）。

## 資料來源說明

本系統使用以下資料來源，全部免費，不需額外申請 Key：

**Yahoo Finance**

提供台股與美股的即時股價、歷史走勢、本益比、殖利率、市值等基本面數據。系統每 60 秒自動刷新一次。台股股票代號需加上 `.TW` 後綴，例如台積電為 `2330.TW`。

**TWSE 台灣證券交易所開放資料**

提供外資買賣超數據，每 2 分鐘刷新一次。資料直接從政府開放資料 API 取得，不需任何帳號或 Key。

**Claude（Anthropic）/ GPT-4o（OpenAI）**

所有 AI 分析功能均由大型語言模型提供。模型本身的知識有截止日期，因此本系統在呼叫 AI 分析時，會先自動從 Yahoo Finance 取得最新股價與財務數據，再一起傳送給 AI，確保分析內容基於即時數字。

## 功能模組說明

### 一、AI 對話理專

這是主要的對話介面，使用者可以用自然的中文詢問任何投資相關問題，AI 會以串流方式逐字回覆，不需要等待全部回應完成。

對話具有記憶功能，同一個 session 中 AI 會記住前面的問答內容，因此可以進行連續性的深入討論，例如先問「分析台積電」，接著問「那停損點應該設在哪裡」，AI 會理解你說的是台積電。

左側欄提供六個快速詢問按鈕，涵蓋今日大盤分析、熱門投資題材、投資組合規劃、基本面指標、技術分析入門、存股策略等常見問題，點擊後直接發送。

自選股區域顯示五檔股票的即時報價，點擊任何一檔可以直接詢問 AI 對該股的分析意見。

右側面板提供快速分析工具，可以選擇個股分析、新聞解讀、組合評估或大盤展望四種模式，輸入內容後由 AI 快速產出結構化回覆。

---

### 二、模組化 AI 提問範本庫

提問品質直接影響 AI 分析的深度與準確度。這個模組收錄了 16 個經過設計的專業提問範本，分為四個類別：

**財報分析類**

包含季報深度分析、年報財務健康度、EPS 成長分析、殖利率存股評估。每個範本都包含完整的分析框架，例如「請分析本益比趨勢、毛利率變化、EPS 年增率、自由現金流」等具體問題清單。

**選股操作類**

包含成長股篩選條件、AI 自動估值分析、產業體質篩選、投資組合配置 Prompt。幫助使用者系統性地評估個股。

**精準進退場類**

包含股價趨勢分析、技術指標買入訊號、停損停利策略設定、回測策略評估。側重在操作時機的判斷。

**整合分析類**

包含個股完整健診報告、產業輪動分析、總經環境投資評估、投資組合整體健診。提供宏觀視角的分析框架。

每個範本可以點擊「套用到對話」，系統會自動帶入對話輸入框，使用者可以進一步修改，填入自己的股票名稱或數字後送出。也可以點擊「複製」，貼到其他 AI 工具使用。

### 三、AI 投資儀表板

儀表板提供視覺化的市場概覽，所有數字均來自即時資料。

最上方的四個 KPI 卡片顯示台積電、聯發科、台加權指數、S&P 500 的即時股價與漲跌幅。

台積電六個月走勢折線圖，資料來自 Yahoo Finance 的每日歷史收盤價，圖表右上角會顯示這段期間的報酬率。

產業配置圓餅圖，依據自選股清單中各股的所屬產業自動計算。

即時股價清單，顯示自選股五檔的最新股價、漲跌幅與本益比，並標示最後更新時間。

台加權指數月績效長條圖，抓取近六個月的月線資料，綠色代表當月上漲，紅色代表下跌。

### 四、AI 自動化估值模組

輸入一檔股票的財務指標，AI 會自動整合即時股價，執行多種估值方法的計算與比較。

**使用方式**

輸入股票代號或名稱，系統會自動查詢即時股價。再填入本益比、股價淨值比、ROE、EPS、預估盈餘成長率，並選擇所屬產業，點擊開始估值分析。

**AI 會提供**

- 本益比法估值（根據歷史 PE 區間計算合理價格）
- 股價淨值比法估值
- 殖利率法估值（適用於高配息股）
- 簡化版 DCF 現金流折現估值
- 與同業估值水準比較
- 綜合合理價格區間（偏低、合理、偏高的分界線）
- 安全邊際評估
- 具體的買入、持有或賣出建議

---

### 五、技術指標分析模組

輸入股票目前的技術指標數值，AI 會解讀這些數字代表的意義並給出操作建議。

**需要輸入的欄位**

股票名稱或代號、目前股價、5 日均線 MA5、20 日均線 MA20、60 日均線 MA60、成交量趨勢（放量上漲、縮量下跌等）、KD 指標的 K 值與 D 值、RSI 數值、近期走勢描述（例如「連三紅、突破前高」）。

系統在呼叫 AI 之前，會先從 Yahoo Finance 抓取該股近 30 日的每日收盤價與成交量，一併提供給 AI 作為參考，讓分析更加準確。

**AI 會提供**

- 均線多空排列判斷（多頭、空頭、整理格局）
- KD 指標解讀（是否超買超賣，有無黃金交叉或死亡交叉）
- RSI 強弱位置評估
- 重要支撐位與壓力位
- 短、中、長期趨勢判斷
- 具體操作策略，包含建議進場點、停損點、目標價

---

### 六、回測分析模組

回測是指將一個交易策略套用到歷史資料上，計算如果過去幾年都按照這個策略操作，報酬率與風險會是如何。

本模組的回測是由 AI 進行模擬分析，不是真正的程式化歷史數據回測，因此結果為參考性質。

**使用方式**

填入策略名稱、回測標的（例如台積電或 0050）、回測期間（近 1 年至近 10 年）、初始資金，以及最重要的買入條件與賣出條件。

買入條件和賣出條件請用文字描述，越具體越好。例如「當 KD 指標 K 值從 20 以下向上突破 D 值，且 RSI 小於 50 時買入」，「當股價跌破 20 日均線，或虧損超過買入價 8% 時賣出」。

**AI 會提供**

- 策略邏輯的合理性評估
- 模擬回測結果，包含預估年化報酬率、最大回撤幅度、勝率
- 策略的優點
- 策略的缺點與容易失效的市場環境
- 改進建議
- 與大盤買進持有策略的比較

---

### 七、投資組合配置

輸入目前的持股明細，AI 會分析組合的健康程度並提供最佳化建議。

**使用方式**

選擇風險承受度（保守型、穩健型、積極型、高積極型）、投資目標（穩健成長、高殖利率收息、長期資本增值、退休規劃）、投資期間，然後在文字框中輸入持股明細，每行填一檔，格式為「代號 名稱 金額或比例」。

**AI 會提供**

- 當前組合健診，包含集中度、多元性、風險評分
- 產業與地區配置的分析
- 建議的最佳化配置比例
- 哪些標的應該減碼，哪些應該增碼，以及原因
- 可以考慮加入的新標的，例如 ETF 或個股
- 預期改善後的風險收益比變化

---

### 八、自動化投資報告

選擇報告期間與重點方向，輸入持股明細，一鍵生成結構化的投資分析報告。

報告包含執行摘要、市場環境回顧、持股績效分析、風險指標評估、重要事件影響、下期展望與策略、調整建議、免責聲明等章節。

生成後可點擊複製全文，貼入 Word 或 Notion 等工具進行整理存檔。

---

### 九、每月重點摘要推播

選擇月份，AI 會生成該月台股市場的重點摘要。

摘要包含大盤表現、產業亮點（前三強勢與前三弱勢產業）、重要財經事件、籌碼動向（外資、投信、自營商的操作方向）、本月最強個股、下月關鍵觀察指標、AI 投資建議重點、風險警示。

頁面底部有「繼續在 AI 討論」按鈕，點擊後會將摘要內容帶入對話模組，可以進一步請 AI 根據這份摘要提供下個月的投資策略建議。

### 十、使用說明與 Python 程式範例

這個模組分為三個子頁籤。

**系統說明**

提供所有功能模組的簡介，適合第一次使用時閱覽。

**Python 範例**

提供三段可直接使用的 Python 程式碼：

第一段是使用 yfinance 套件下載台股歷史股價的範例，並計算 MA5、MA20、MA60 三條均線。

第二段是計算 KD 隨機指標的函式，並示範如何找出黃金交叉訊號。

第三段是本益比估值計算的函式，輸入 EPS 與歷史 PE 區間，自動計算合理價格區間。

初學者可以複製這些程式碼，安裝對應套件後在 Jupyter Notebook 或本地 Python 環境中執行，作為自動化分析的起點。

**操作流程**

提供建議的每日投資工作流程，包含開盤前準備、盤中觀察、收盤後檢視、每週組合健診、月底生成月報的完整作業流程。

同時提供四個提高 AI 分析品質的提問技巧：提供具體背景資訊、說明投資目標與風險承受度、善用範本庫再客製化、連續追問深化分析。

## 專案檔案結構

```
ai-investment/
├── server.js          後端伺服器，處理所有 API 請求
├── package.json       Node.js 相依套件設定
├── render.yaml        Render 部署設定
├── .gitignore         Git 忽略清單
├── README.md          本說明文件
└── public/
    └── index.html     前端介面，所有 UI 與 JavaScript
```

後端 `server.js` 負責：與 Yahoo Finance 取得即時資料、向 TWSE 取得外資數據、呼叫 Claude 或 GPT-4o 的 API、提供快取機制避免重複呼叫。

前端 `index.html` 是一個單頁應用程式，所有畫面切換都在瀏覽器端完成，不需要重新載入頁面。
